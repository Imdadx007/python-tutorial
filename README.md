# 🐍 GitHub Profile Setup Script

This is a Python script that collects user information and generates a structured GitHub profile data file in JSON format.

It takes basic details like username, name, email, bio, skills, and number of projects, then saves everything into a `github_profile.json` file.

---

## 🚀 Features
- Collects user input from terminal
- Stores profile data in structured JSON format
- Handles multiple skills using comma-separated input
- Simple and beginner-friendly script

---

## 🛠 Technologies Used
- Python
- JSON file handling

---

## 📂 How It Works
1. User enters GitHub details (username, name, email, etc.)
2. Skills are entered as comma-separated values
3. Script processes and formats the data
4. Data is saved into `github_profile.json`
